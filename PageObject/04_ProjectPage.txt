#******************** Test Cases *************************

*** Test Cases ***

Create blank project
- Navigate to ProjectUrl.
- Click on the New project button.
- Select blank project.
- Input Project_Name as project name.
- Select any group in project URL.
- Click on the Create project button.
- Verify that you are redirected to your project and that there is a message "Project 'Project_Name' was successfully created." at top of the page.

Read blank project
- Upon creation of the new blank project Project_Name you are automatically redirected to it.
- Verify you are able to view the README.md file that was automatically generated during project creation.

Update blank project
- Navigate to ProjectUrl.
- Select the Project_Name project.
- One the left side navigate to Settings and General.
- Change Project name to Project_Name_2.
- Ckick on the Save changes button.
- Verify that there is a message "Project 'Project_Name_2' was successfully updated." at top of the page.
- Navigate to ProjectUrl and verify the project name has been updated.

Delete blank project
- Navigate to ProjectUrl.
- On the far right click on the triple dots on the same row as Project_Name_2.
- From the context menu select Delete.
- Follow instructions and confirm deletion.
- Verify that the project is no longer listed at ProjectUrl.

Create template project
- Navigate to ProjectUrl.
- Click on the New project button.
- Select Create from template.
- Select Sample GitLab Project.
- Input Project_Name as project name.
- Select any group in project URL.
- Click on the Create project button.
- Verify that you are seeing "Import in progress" message at top of the page.
- Verify that you are after a moment redirected to your project and that there is a message "Project was successfully imported." at top of the page.

Read template project
- Upon creation of the new template project Project_Name you are automatically redirected to it.
- On the left side navigate to Plan and Issues.
- Verify that you can see numerous Issues.
- On the left side navigate to Code and Merge requests.
- Verify that you can see numerous Merge requests.
- On the left side navigate to Code and Branches.
- Verify that you can see numerous Branches.
- On the left side navigate to Code and Commits.
- Verify that you can see at least one Commit.

Update template project
- Navigate to ProjectUrl.
- Select the Project_Name project from under Member category.
- One the left side navigate to Settings and General.
- Change Project name to Project_Name_2.
- Ckick on the Save changes button.
- Verify that there is a message "Project 'Project_Name_2' was successfully updated." at top of the page.
- Navigate to ProjectUrl and verify the project name has been updated. The project is listed under Member category.

Create blank project with project name that is in use
- Navigate to ProjectUrl.
- Click on the New project button.
- Select blank project.
- Input Project_Name_2 as project name.
- Select same group as in Create template project.
- Click on the Create project button.
- Verify that you receive a following message at top of the page:
The form contains the following errors:
* Project namespace name has already been taken
* Name has already been taken

Delete template project
- Navigate to ProjectUrl.
- On the far right click on the triple dots on the same row as Project_Name_2.
- From the context menu select Delete.
- Follow instructions and confirm deletion.
- Verify that the project is no longer listed at ProjectUrl under the Member category.
