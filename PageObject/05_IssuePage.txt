# NOTE: Please Remove your login credential before uploading Assignments

ProjectUrl = https://gitlab.com/dashboard/projects
Issue_Title_1 = Testing
Issue_Title_2 = Testing further
Issue_Description = This is for testing issue creation functionality.
Issue_Description_Addition = This is for testing editing functionality.
Issue_Comment = This is a comment to test comment functionality.


#******************** Test Cases *************************

*** Test Cases ***

Create Issue
- Navigate to ProjectUrl.
- Select your test project.
- On the left side navigate to Issues, or Plan and then Issues if Issues is not a pinned item.
- Click on the New item button.
- Input Issue_Title_1 as title.
- Input Issue_Description as the issue's description.
- Click on the Create issue button.
- Verify that the issue appears in the open issues listing.

Read Issue
- Open the issue created in the previous test case.
- Verify that the issue opens and displays Issue_Description as its comment.

Update Issue
- Open the issue created in the first test case.
- Click the Edit button.
- Change the issue's title to Issue_Title_2.
- Add Issue_Description_Addition to the issue's description after a newline.
- Click the Save changes button.
- Verify that the title and description were updated.

Add Comment
- Open the issue created in the first test case.
- Add Issue_Comment to the textbox below Activity.
- Click on the Comment button.
- Verify that the comment was added to the issue.

Create Issue #2
- Ensure you still have the same project open as in the first test case.
- On the left side navigate to Issues, or Plan and then Issues if Issues is not a pinned item.
- Click on the New item button.
- Input Issue_Title_2 as title.
- Input Issue_Description as the issue's description.
- Click on the Create issue button.
- Verify that the issue appears in the open issues listing.

Close Issue
- Open the issue that was created in the first test case.
- Click the Close issue button.
- Verify that the issue was closed.

Delete Issue
- Open the issue that was created in the fifth test case.
- Click on the triple dots at the top right corner.
- Select Delete issue.
- Confirm deletion.
- Verify that the issue was deleted and does not show in All issues.
